# workplace
Electron Desktop app for Workplace

# prerequisites
- node:
   - `node -v`
- npm:
   - `npm -v`

# launching
- `npm start`

# building
- macOS arm64
   - `npm run make -- --arch=arm64 --platform=darwin`
